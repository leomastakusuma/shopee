# Shopee Tax Calculator

### GETTING STARTED
```sh
$ git clone https://github.com/leomastakusuma/shopee.git
$ cd shopee
```

### RUN with Docker 
```sh
$ docker-compose build 
$ docker-compose up -d 
```
